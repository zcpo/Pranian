# **App Name**: Pranian

## Core Features:

- Content Library: A searchable library of yoga and meditation content, including articles, videos, and audio tracks.
- Free Tier Trial: Offer users a limited-time free trial to access premium content. Admin can configure the trial duration.
- Subscription Management: Enable users to subscribe to a paid tier for full access to all content at $3.99/month.
- E-commerce Store: Sell yoga and meditation-related products such as podcasts, merchandise, events, books, videos, and online learning courses.

## Style Guidelines:

- Background color: Pure black (#000000) for a modern feel.
- Text color: Pure white (#FFFFFF) for contrast and readability.
- Accent color: A medium shade of violet (#A073FF) will be used for highlighting interactive elements, creating a luxurious, trendy, cutting edge appearance.
- Font: 'Inter' sans-serif font will be used as both headline and body font; Note: currently only Google Fonts are supported.
- Use simple, elegant icons in violet to represent different content categories and actions.
- Design the layout to be clean and intuitive, inspired by Apple's minimalist aesthetic, with clear visual hierarchy and ample whitespace.
- Incorporate subtle animations for transitions and interactions to enhance user engagement without being distracting.
